# 问卷设计

> 一个题目设计组件，多种类型题目设计，更快更好更完善的完成问卷设计

**UI组件基于elementUI`2.13.2`**
**排序组件依赖vuedraggable`^2.24.3`**

## 关于题目类型
- [选择题](单项选择、多项选择、下拉框选择、级联选择)
- [填空题](单项填空、多项填空)
- [评分题](量表题、NPS量表、评分单选、评分多选、矩阵量表)
- [个人信息题](姓名、手机、邮箱、性别、日期、时间、地址城市、高校)
- [其他题型](文件上传、矩阵选择、矩阵打分、矩阵填空)
